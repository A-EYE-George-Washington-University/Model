# AEYE
## Introduction 
Multi-class image classifer for Fundus Opthamologic Medical Imagings

Detects the following disease:
* Normal
* Diabetic Retinopathy 
* Cataract
* Glaucoma
* Age Related Macular Degeneration
* Patholical Myopia
